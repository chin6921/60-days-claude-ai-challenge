# 🥗 Day 9 – Building NutriScope Pro with Claude

## 📌 Overview

Today I explored AI-assisted web application development by building **NutriScope Pro**, a nutrition tracking platform using Claude. Starting with a detailed MVP prompt, I then enhanced the application with advanced features through iterative prompting.

---

## 🎯 Objective

- Build a complete nutrition tracking application.
- Practice iterative prompt engineering.
- Enhance an existing application with advanced functionality.
- Explore frontend application development using AI.

---

## 📝 Prompt 1 – Build MVP

The first prompt generated the foundation of **NutriScope Pro**.

### Features

- User Health Profile
- Food Logger
- Nutrition Calculator
- Daily Energy Targets
- Macro & Micronutrient Tracking
- Personalized Recommendations
- Interactive Dashboard
- Premium Dark SaaS UI
- Mobile Responsive Design
- Chart.js Integration
- Single HTML File (No Backend)

---

## 📝 Prompt 2 – Application Enhancement

The second prompt expanded the application with additional capabilities.

### New Features Added

- CSV Food Import
- Expanded Food Database
- Additional Micronutrients
- 2-Day Meal Planner
- Nutrition Risk Analysis
- Educational Disclaimer
- Nutrition Sources
- Improved Charts & Analytics
- Advanced Personalized Recommendations

---

## 🛠️ Skills Practiced

- Prompt Engineering
- Frontend Development
- HTML
- Responsive Web Design
- Dashboard UI/UX
- Data Visualization
- AI-Assisted Software Development

---

## 💡 Key Learnings

- Breaking large projects into multiple prompts improves the final result.
- Detailed prompts help Claude generate cleaner and more organized applications.
- Iterative prompting allows continuous feature enhancement without rebuilding from scratch.
- AI can significantly speed up frontend prototyping and dashboard development.

---

## 🎯 Reflection

This project showed how prompt engineering can transform a simple idea into a professional, feature-rich web application. Rather than writing everything manually, I focused on designing better prompts and refining the application through multiple iterations.
