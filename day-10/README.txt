## 🚀 Quick Start

### Option 1: Deploy Instantly (Recommended)

**Netlify (5 seconds):**
1. Download `portfolio.html`
2. Drag & drop into [Netlify](https://app.netlify.com/drop)
3. Get a live URL instantly

**GitHub Pages:**
1. Fork this repository
2. Enable GitHub Pages in settings
3. Access your portfolio at `yourusername.github.io`

### Option 2: Local Development

```bash
# Clone the repository
git clone https://github.com/yourusername/portfolio.git

# Navigate to directory
cd portfolio

# Open in browser
open portfolio.html
# or
start portfolio.html
```

No build process, no dependencies — just open in your browser!

## 🎨 Customization

### Update Personal Information
Open `portfolio.html` and replace these placeholders:

```html
<!-- Hero Section -->
<h1>Your Name</h1>
<p>Your Professional Title</p>

<!-- Contact Section -->
<a href="mailto:your.email@example.com">your.email@example.com</a>
<a href="https://github.com/yourusername">GitHub Link</a>
<a href="https://linkedin.com/in/yourprofile">LinkedIn Link</a>
```

### Modify Color Scheme
Edit the CSS variables in the `<style>` section:

```css
:root {
    --primary: #d946ef;        /* Neon Purple - Change this! */
    --secondary: #0f172a;      /* Dark Navy */
    --accent: #1e293b;         /* Medium Dark */
    --text-dark: #e2e8f0;      /* Light Text */
    --text-light: #1a1a1a;     /* Dark Text */
}
```

### Add Your Projects
Update project cards in the **Projects Section**:

```html
<h3>Your Project Title</h3>
<p>Project description...</p>
<a href="https://github.com/yourrepo">View Project</a>
```

### Update Skills
Add or remove skills in the **Skills Section**:

```html
<span class="tech-badge">Your Skill</span>
```
